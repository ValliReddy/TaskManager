# ✅ TaskMaster - Smart Task Management App

**TaskMaster** is a full-stack task management web application designed for users to stay organized and productive. Built with **HTML, CSS, Flask, and MongoDB**, it offers a clean interface, secure authentication, and powerful backend logic for managing tasks efficiently.

---

## 🚀 Live Demo

- **Project link:** https://taskmanager-2byu.onrender.com/

---

## ✨ Key Features

### 🔐 Secure User Authentication
- Login and registration with hashed password storage 🔒  
- User sessions for protected access to personal task data  

### 📝 Task Management
- Create, update, and delete tasks  
- Track progress and mark tasks as completed  
- Intuitive UI for easy interaction  

### 🔁 RESTful APIs
- Seamless client-server communication  
- Clean and scalable endpoint structure  

### ☁️ Cloud Deployment
- Hosted on **AWS EC2** for high scalability and performance  
- Stable and reliable for production environments  

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS  
- **Backend:** Flask (Python)  
- **Database:** MongoDB  
- **Deployment:** AWS EC2  
- **Security:** Password hashing with Werkzeug
